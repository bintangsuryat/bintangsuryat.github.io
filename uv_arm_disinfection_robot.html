UV DISINFECTION ARM ROBOT

- Develop UAV robot system for mission to find & extinguish a flame in the field
- This is project for UAV robotic contes 2016 (Kontes Robot Terbang Indonesia (KRTI) 2016) Vertical Take Off Landing Division

Mechanics & manufacture:

• ACP Plate • Hollow Aluminium • Extrusion Aluminium

Hardware:

• Ardupilot Apm 2.8 as main (master)flight controller for autopilot system control • Arduino Nano 328p as slave controller for giving algorithm instructions to flight controller & control payload system with multiplexing method. • Pixy camera as image processing censor for flame detect • GPS Neo M8N for giving coordinate navigation

Communication:

• Telemetry 3dr, to give flight information to ground control station • Radio module 2.4 ghz, as comunication system with remote control • Turnigy Remote control

Propultion:

• 4 motor brushless 150 kv + 11.5 inch propellers as actuator • 4 esc 40A as driver motors • 4 cell 5200mAh lippo battery as main source power • 12 V dc water pump
