UAV Pick and Drop Kit Survival Robot 2018

- This is project for UAV robotic contes 2018 (Kontes Robot Terbang Indonesia (KRTI) 2018) Vertical Take Off Landing Division

Mechanics & manufacture:

• Quadcopter Configuration • Alumunium frame material manufacture • 2 servos to drop logistic mechanism • Magnet induction mechanism to pick a kit logistic

Hardware:

• Pixhawk as main (master)flight controller for autopilot system control • Mini PC to giving autonomous algorithm instructions to flight controller & control payload system • Slave Microcontroller Arduino Nano for actuator control • Logitech web cam as image processing censor • GPS Neo M8N for giving coordinate navigation

Communication:

• Tp link wifi router 5.8 Ghz, to give flight information to ground control station • Radio module 2.4 Ghz, as comunication system with remote control • Taranis Remote control

Propultion System:

• 4 motor brushless T motor 400 kv + 11.5 ins propellers as actuator • 4 esc 40A as driver motors • 2 cell 2200 mAh lippo battery for mini pc • 6 cell 5200 mAh lippo battery as main source power
