UAV Pick and Drop Kit Survival Robot 2017

- Develop UAV robot system for mission to find & drop logistic on a coordinate in a field
- This is project for UAV robotic contes 2017 (Kontes Robot Terbang Indonesia (KRTI) 2017) Vertical Take Off Landing Division

Mechanics & manufacture:

• H- Octacopter Configuration • Alumunium frame manufacture • 2 servos to drop logistic mechanism • Magnet induction mechanism to pick a kit logistic

Hardware:

• Pixhawk as main (master)flight controller for autopilot system control • Mini PC to giving autonomous algorithm instructions to flight controller & control payload system • Slave Microcontroller Arduino Nano for actuator control • Logitech web cam as image processing censor • GPS Neo M8N for giving coordinate navigation

Communication:

• Tp link wifi router 5.8 Ghz, to give flight information to ground control station • Radio module 2.4 Ghz, as comunication system with remote control • Taranis Remote control

Propultion:

• 8 motor brushless rc timer 150 kv + 11.5 ins propellers as actuator • 8 esc 40A as driver motors • 4 cell 8000mAh lippo battery as main source power
